# dise-o-de-vigas-simples-matlab
Qué hace este programa?
🔹 Recibe múltiples solicitaciones (momentos últimos) a través de un vector, evaluándolos iterativamente.
🔹 Verifica si la sección transversal resiste calculando el momento máximo normativo.
🔹 Si la viga no resiste, detiene el cálculo y advierte que se deben incrementar las dimensiones.
🔹 Si el cálculo es factible, optimiza el área de acero requerida y la compara automáticamente con la cuantía mínima por norma.
What does this program do?
🔹 Takes multiple ultimate moments through a vector, evaluating them iteratively.
🔹 Checks if the cross-section can withstand the load by calculating the maximum nominal moment.
🔹 If the beam cannot resist, it stops the calculation and warns that the dimensions must be increased.
🔹 If feasible, it calculates the required steel area and automatically compares it with the minimum reinforcement required by code.
